❌ PROBLEM:
D1 database does not support SQLite dot commands like:
- .tables
- .schema
- .describe
- .help
etc.

✅ SOLUTION:
Use standard SQL commands instead:

INSTEAD OF:           USE THIS:
-----------           --------
.tables               SELECT name FROM sqlite_master WHERE type='table';
.schema tablename     SELECT sql FROM sqlite_master WHERE name='tablename';
.describe table       PRAGMA table_info(tablename);

✅ WORKING COMMANDS FOR D1:
1. List tables:
   SELECT name FROM sqlite_master WHERE type='table';

2. Check if table exists:
   SELECT COUNT(*) FROM sqlite_master WHERE type='table' AND name='dompet_santri';

3. Get table structure:
   SELECT sql FROM sqlite_master WHERE name='dompet_santri';

4. Count records:
   SELECT COUNT(*) FROM dompet_santri;

5. Select data:
   SELECT * FROM dompet_santri LIMIT 5;
